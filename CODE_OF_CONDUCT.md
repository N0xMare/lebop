# Code of Conduct

This project follows the [Contributor Covenant v2.1][covenant]. By
participating — opening issues, commenting, contributing code or docs —
you agree to uphold it.

## The short version

- Be respectful. Disagreement is fine; personal attacks aren't.
- Assume good faith. Most miscommunication isn't malice.
- Keep technical critique focused on the work, not the person.
- No harassment, doxxing, or discriminatory remarks. Period.

## Reporting

If you experience or witness behavior that violates the Covenant, contact
the maintainer through their GitHub profile
(`https://github.com/N0xMare`). Reports are handled privately. The
maintainer reserves the right to remove comments, close issues, and ban
contributors whose behavior is incompatible with the Covenant.

## Scope

This applies in all project spaces — issues, pull requests, discussions,
and any community channels — and to representations of the project in
public spaces.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
